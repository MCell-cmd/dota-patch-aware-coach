# Evidencia

Guardar aqui solo informacion verificable y fechada.

Formato recomendado por archivo:

```md
# <fuente o tema>

- Fecha de consulta:
- URL:
- Tipo de fuente:
- Resumen:
- Citas cortas relevantes:
- Que problema confirma:
- Que no confirma:
- Riesgos o limites:
```

No guardar tokens, cookies, credenciales, mensajes privados ni datos personales.
