# Scoring Reference

Initial pick score:

```text
score = comfort + lane_matchup + team_synergy + counter_value + patch_value - execution_risk
```

Recommended starting weights:

| Factor | Weight |
| --- | ---: |
| Comfort | 25 |
| Lane matchup | 20 |
| Team synergy | 20 |
| Counter value | 15 |
| Patch value | 10 |
| Execution risk | -10 |

Draft checks:

- Missing stun.
- Missing initiation.
- Missing wave clear.
- Missing tower damage.
- Missing save or sustain.
- Poor Roshan damage.
- Too much physical or magical damage.
- No answer to enemy win condition.
- Draft needs early close vs late scaling.
